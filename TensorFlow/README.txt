This folder contains scripts from Youtube tutorials on how to use TensorFlow for machine learning and deep learning. 
The following scripts were produced during walkthroughs and tutorials of the "TensorFlow 2.0 Complete Course - Python Neural Networks for Beginners Tutorial" by freeCodeCamp.org
and can be found at the following link: https://www.youtube.com/watch?v=tPYj3fFJGjk . The supporting documentation is listed next to each filename. This tutorial was completed in
Google Colab, most of which originated from the TensorFlow documentation https://www.tensorflow.org/tutorials
